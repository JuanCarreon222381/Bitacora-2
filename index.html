Bitácora de Desarrollo Web

Mobile First

Ideas Principales

  • Se diseña primero para dispositivos móviles y luego se adapta a pantallas más grandes.
  • Optimiza el rendimiento en dispositivos con menor capacidad.
  • Favorece una mejor experiencia de usuario al priorizar el contenido esencial.
  • Se basa en el uso de media queries para ampliar el diseño a dispositivos más grandes.

Definición

El enfoque "Mobile First" prioriza el diseño para dispositivos móviles antes que para pantallas más grandes. Esto mejora la accesibilidad y el rendimiento en dispositivos con menor capacidad.

Módulo Flexbox - Padre

Ideas Principales

  • Flexbox permite organizar elementos de forma flexible dentro de un contenedor.
  • Facilita la alineación de elementos en diferentes direcciones (filas o columnas).
  • Se usa para distribuir el espacio de manera eficiente, incluso cuando los tamaños de los elementos son desconocidos.
  • Controla la alineación horizontal y vertical de los elementos hijos.

Definición

Flexbox es un modelo de diseño CSS que facilita la alineación y distribución de elementos en un contenedor flexible.

Módulo Flexbox - Hijo

Ideas Principales

  • Los elementos hijos pueden crecer o reducirse dinámicamente según el espacio disponible.
  • Se puede controlar su tamaño inicial y cómo se ajustan dentro del contenedor.
  • Ayuda a crear interfaces flexibles y responsivas sin necesidad de tamaños fijos.

Definición

Los elementos hijos dentro de un contenedor Flexbox pueden alinearse de manera flexible utilizando propiedades específicas.

Imágenes Responsivas

Ideas Principales

  • Deben adaptarse automáticamente a diferentes tamaños de pantalla.
  • Se pueden utilizar diferentes formatos según la resolución del dispositivo.
  • picture y srcset permiten optimizar la carga de imágenes para cada pantalla.
  • Evita el uso de imágenes fijas que puedan afectar la experiencia de usuario.

Definición

Las imágenes responsivas se adaptan automáticamente al tamaño del dispositivo sin perder calidad.

SVG

Ideas Principales

  • Son gráficos basados en vectores, lo que permite escalarlos sin perder calidad.
  • Ocupan menos espacio en comparación con imágenes en formatos como PNG o JPG.
  • Se pueden modificar y animar directamente con CSS y JavaScript.
  • Son ideales para iconos, ilustraciones y gráficos en sitios web modernos.

Definición

Los gráficos SVG son imágenes basadas en vectores que pueden escalarse sin perder calidad.

Variables CSS

Ideas Principales

  • Permiten reutilizar valores de colores, tamaños y otros estilos sin repetir código.
  • Se declaran en :root para que estén disponibles en todo el documento.
  • Facilitan el mantenimiento y la personalización del diseño.
  • Se pueden cambiar dinámicamente con JavaScript para temas oscuros o personalizados.

Definición

Las variables CSS permiten almacenar valores reutilizables, como colores o tamaños de fuente.

Tipografía Responsiva

Ideas Principales

  • La tipografía debe adaptarse a diferentes tamaños de pantalla para una mejor legibilidad.
  • Se recomienda el uso de unidades relativas como em, rem, % o vw.
  • Clamp() permite establecer un tamaño mínimo, preferente y máximo en una sola línea.
  • Mejora la experiencia de usuario en dispositivos móviles sin necesidad de media queries.

Definición

El texto debe adaptarse a distintos dispositivos mediante unidades como em, rem o vw.
